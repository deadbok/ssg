Markdown
********

Markdown_ allows intuitive, and
easy to read text, to be converted to HTML.

Static Site Generator provides some extensions to Markdown, along with the 
extensions of Python-Markdown_.

These extensions_ are enabled by default:

+ Extra extensions
+ Meta-data extension
+ Table of Contents extension

.. _Markdown: http://daringfireball.net/projects/markdown/
.. _Python-Markdown: https://pythonhosted.org/Markdown/
.. _extensions: https://pythonhosted.org/Markdown/extensions/index.html
